Find esports teams and players on CS2.TEAM.

## Requirements

Make sure you have Node and NPM installed:

```bash
node -v
# v22.10.0
```

```bash
npm -v
# 10.9.0
```

## Installation

Run the commands:

```bash
npm install
```

```bash
npm run dev
```
